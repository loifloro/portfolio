What I Learned

We built this project when I was second year college and at that time I have very little idea how to make a web application. During the development phase of this project, I learned to use the grid system of Bootstrap

Even though the overall UI of this project isn't proffesionally neat, we manage to create an atleast usable design. In fact, our group got the highest score for this final project in Advanced Database System

My laptop at that time is super slow so that I can only do minimal task such as designs, improvement of user interface. After creating this project, my ideas of what more to learn have expanded and I think this project really opened doors for me to strive on gaining infinite knowledge. I hope that holds up to this day.
