Process

What I Learned

In this project, it is my first to fully design a User Interface. This is a partial fulfillment for my subject - System Analysis and Design. Our main task was to create a high fidelity design of an e-commerce and this was my group's submission. On creating this project I learned to choose fonts and how to it effieciently.

I also learned how to combine color scheme to effectively visualize the elements in the design. Lastly, while developing the prototype, I learned what are the necessary elements that an e-commerce website should have and should emphasize.
